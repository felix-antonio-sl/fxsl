# FXSL Namespace - Knowledge Engineering

Namespace personal de Felix Sanhueza para proyectos de ingeniería de conocimiento.

## Estructura

- `agents/` - Agentes KODA especializados (modelador-mbt, arquitecto-categorico)
- `knowledge/` - Base de conocimiento (category theory, methodologies)
- `tooling/` - Workflows, rules, profiles específicos
