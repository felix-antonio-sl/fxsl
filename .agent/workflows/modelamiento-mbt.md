# Workflow: Modelamiento Basado en Tensiones (MBT)

Este workflow te guía a través de un proceso de modelamiento explícito y reflexivo.

## Fases

### 1. Diagnosticar Contexto
Evalúa las condiciones del trabajo:
- Recursos: ¿Cuánto tiempo? ¿Solo o en equipo?
- Propósito: ¿Explorar o especificar? ¿Para humanos o máquinas?
- Dominio: ¿Tema conocido o nuevo? ¿Estable o volátil?
- Cultura: ¿Ambiente formal o informal?

### 2. Establecer Postura
Define cómo vas a modelar:
- Criterios de inclusión/omisión
- Audiencia y nivel de detalle
- Estrategia top-down o bottom-up
- Criterios de completitud

### 3. Navegar Tensiones
Para cada decisión de modelamiento, hacer explícita la tensión:
- ¿Entidad o Evento?
- ¿Concreto o Abstracto?
- ¿Todo o Partes?
- ¿Estático o Dinámico?
- etc.

Documenta cada decisión con justificación.

### 4. Validar
- Verificar coherencia interna
- Evaluar correspondencia con realidad
- Decidir si iterar o entregar

## Uso

Puedes invocar este workflow en cualquier momento con `/modelamiento-mbt`.

El asistente te hará preguntas para:
1. Diagnosticar tu contexto
2. Establecer tu postura de modelamiento
3. Identificar tensiones en tus decisiones
4. Validar tu modelo final
